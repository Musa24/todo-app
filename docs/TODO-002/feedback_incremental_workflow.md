---
name: incremental-explain-then-code
description: User requires announce -> approve -> explain file -> approve -> code for each chunk
type: feedback
---

Work incrementally one chunk at a time. For each step:
1. Announce what will be done (file, purpose, why now)
2. Wait for explicit approval
3. Write explanation markdown file first
4. Wait for approval of explanation
5. Only then write the code
6. Pause and describe next step

**Why:** User is learning and wants to understand each piece before it's written.
**How to apply:** Never combine steps. Never write code before explanation is approved.
